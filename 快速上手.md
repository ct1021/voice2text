# voice2text · 快速上手（新手向）

> 拿到压缩包的朋友看这一份就够了。完整说明见 `README.md`。
> 一句话：**按住一个键说话，松手，你说的话就被转成文字、AI 清干净、自动粘到光标处。**

---

## 一、它能干什么

在任何能打字的地方（对话框、搜索框、聊天、Claude Code…），不用打字——
按住一个键说话，松手，干净的文字自动出现在光标处。语音识别在你电脑本地跑，声音不上传。

---

## 二、跑起来要什么

- Windows 10 / 11，或 macOS
- **Python 3.11**（去 python.org 下载安装，安装时勾选「Add Python to PATH」）
- 内存建议 16GB 以上
- （可选）Node.js —— 只有用 `claude-sdk` 后端才需要

---

## 三、三步跑起来

**Windows：**

1. 把压缩包解压到一个**纯英文路径**的文件夹（别放中文路径 / 桌面带中文的目录）
2. 进文件夹，右键 `install.ps1` →「使用 PowerShell 运行」（装依赖，等它跑完）
3. 双击 `start.bat` 启动

**macOS：**

1. 解压
2. 终端 `cd` 进文件夹，依次运行：
   ```bash
   chmod +x install.sh run.sh
   ./install.sh
   ./run.sh
   ```
3. 首次运行去「系统设置 → 隐私与安全性 → 辅助功能」给终端 / Python 授权（全局热键要用）

> 首次启动会自动下载语音模型（约 1.5GB），耐心等。

---

## 四、怎么用

1. 启动后桌面出现一个**悬浮小球**
2. 把鼠标光标点进任何一个输入框
3. **按住热键说话**（Windows 默认 `CapsLock`），说完**松手**
4. 等一两秒，干净的文字自动粘进去
5. 小球颜色 = 状态：灰空闲 / 红录音 / 橙处理 / 紫出错
6. **单击小球**看历史，**右键小球**打开菜单（含「整理成 Markdown」模式）

---

## 五、要不要 API key？

- **有 Claude Code 订阅** → 默认就能用，不用填 key
- **没有订阅** → 想要 AI 清洗，把 `.env.example` 复制成 `.env`，填入你自己的
  `DEEPSEEK_API_KEY`（DeepSeek 又快又便宜），再在 `config.toml` 里把
  `[ai] backend` 改成 `openai-compatible`
- **什么都不想配** → `config.toml` 里 `[ai] backend = "none"`，只转写不清洗，也能用

> 压缩包里**不含任何 API key**，key 是你自己的，填在你自己的 `.env` 里。
